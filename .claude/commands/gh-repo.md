# GitHub Repository Management

Available operations:

1. **List repositories**
   - User repositories (authenticated user)
   - Organization repositories
   - Filter by:
     * Visibility (public, private, all)
     * Type (owner, member, all)
     * Sort (created, updated, pushed, full_name)

2. **Get repository details**
   - Owner/repo name
   - View full repository information

3. **Create repository**
   - Repository name
   - Description
   - Visibility (public/private)
   - Initialize with README
   - .gitignore template
   - License

4. **Update repository**
   - Repository name
   - Description
   - Homepage URL
   - Visibility
   - Default branch
   - Settings (issues, wiki, projects)

5. **Delete repository**
   - Repository name
   - Permanent deletion (requires confirmation)

6. **Fork repository**
   - Source repository (owner/repo)
   - Organization (optional)

7. **Clone repository**
   - Repository URL
   - Local path
   - Branch (optional)

8. **Archive repository**
   - Mark repository as archived
   - Read-only state

9. **Get repository topics**
   - List topics/tags

10. **Update repository topics**
    - Add/remove topics

11. **Get repository languages**
    - Language breakdown

12. **Get repository contributors**
    - List all contributors
    - Contribution statistics
