# ZeroDB Operations Index

Complete list of ZeroDB slash commands:

## Core Database Operations
- **/zerodb-projects** - Project management (create, list, update, delete, restore, stats)
- **/zerodb-tables** - Table operations (create, list, CRUD on rows)
- **/zerodb-sql** - Direct SQL query execution with security validation

## Vector & AI Operations
- **/zerodb-vectors** - Vector operations (upsert, search, quantum compression)
- **/zerodb-embeddings** - Generate 384-dim embeddings, embed & store, semantic search
- **/zerodb-semantic** - Advanced semantic search with hybrid scoring
- **/zerodb-memory** - AI agent memory (store, search persistent context)

## Storage & Streaming
- **/zerodb-files** - File operations (upload, download, S3-compatible storage)
- **/zerodb-events** - Event streaming with RedPanda (publish, subscribe)

## AI/ML Data Collection
- **/zerodb-rlhf** - RLHF data collection (feedback, preferences, export)
- **/zerodb-agents** - Agent logging (track actions, performance, traces)

## PostgreSQL Management
- **/zerodb-postgres** - Dedicated PostgreSQL instances (provision, scale, backup)
