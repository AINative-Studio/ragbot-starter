# GitHub Commit Management

Available operations:

1. **List commits**
   - Repository (owner/repo)
   - Filters:
     * Branch/SHA
     * Author
     * Since date
     * Until date
     * Path (specific file)

2. **Get commit details**
   - Repository
   - Commit SHA
   - View full commit information

3. **Compare commits**
   - Repository
   - Base commit SHA
   - Head commit SHA
   - View differences

4. **Get commit diff**
   - Repository
   - Commit SHA
   - View file changes

5. **List commit comments**
   - Repository
   - Commit SHA

6. **Create commit comment**
   - Repository
   - Commit SHA
   - Comment text
   - Line number (optional)
   - Path (optional)

7. **Get commit status**
   - Repository
   - Commit SHA
   - View CI/CD status

8. **Create commit status**
   - Repository
   - Commit SHA
   - State (pending, success, error, failure)
   - Description
   - Target URL
   - Context

9. **List statuses for commit**
   - Repository
   - Commit SHA

10. **Get combined status**
    - Repository
    - Commit SHA
    - Aggregate all statuses

11. **Create commit**
    - Repository
    - Message
    - Tree SHA
    - Parent commit SHA(s)
    - Author info

12. **Get commit parents**
    - Repository
    - Commit SHA

13. **List branches for commit**
    - Repository
    - Commit SHA
    - Find which branches contain commit
