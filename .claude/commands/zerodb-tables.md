# ZeroDB Table Management

Available operations:

## Table Operations
1. **Create table**
   - Table name
   - Description
   - Schema definition (column names and types)

2. **List all tables**
   - View all tables in the current project

3. **Get table details**
   - View schema and metadata for a specific table

## Row Operations (CRUD)
4. **Create row**
   - Table name
   - Row data (JSON object)

5. **List rows**
   - Table name
   - Pagination (skip, limit)

6. **Get row**
   - Table name
   - Row ID

7. **Update row**
   - Table name
   - Row ID
   - Updated data

8. **Delete row**
   - Table name
   - Row ID
