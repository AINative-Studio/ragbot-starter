# ZeroDB Project Management

Available operations:

1. **List all projects**
   - Show all projects with their details (name, tier, status, database enabled)

2. **Create new project**
   - Project name
   - Description
   - Tier (free, pro, scale, enterprise)
   - Database enabled (true/false)

3. **Get project details**
   - Specify project ID to get detailed information

4. **Update project**
   - Update name, description, or tier

5. **Delete project**
   - Soft delete (can be restored) or hard delete (permanent)

6. **Restore deleted project**
   - Restore a previously soft-deleted project

7. **Get projects summary**
   - View aggregated statistics across all projects
   - Total projects, projects by tier, total usage, active projects
