# GitHub Operations Index

Complete list of GitHub slash commands:

## Repository Management
- **/gh-repo** - Repository operations (create, list, update, delete, fork, clone)

## Issue Tracking
- **/gh-issues** - Issue management (create, update, comment, label, assign)

## Pull Requests
- **/gh-pr** - PR operations (create, review, merge, comment, request changes)

## Version Control
- **/gh-branches** - Branch operations (create, delete, protect, merge, compare)
- **/gh-commits** - Commit operations (list, compare, comment, status)

## Code Management
- **/gh-files** - File operations (get, create, update, delete, download)

## Discovery
- **/gh-search** - Search operations (repos, code, issues, users, commits)

## CI/CD
- **/gh-actions** - GitHub Actions (workflows, runs, artifacts, secrets)

## Releases
- **/gh-releases** - Release management (create, update, assets, notes)

## Social & Collaboration
- **/gh-users** - User & org operations (profiles, followers, starred repos)
- **/gh-gists** - Gist management (create, update, fork, star)

## Integrations
- **/gh-webhooks** - Webhook management (create, test, deliveries)
