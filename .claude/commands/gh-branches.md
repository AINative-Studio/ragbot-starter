# GitHub Branch Management

Available operations:

1. **List branches**
   - Repository (owner/repo)
   - Filter protected branches
   - Sort options

2. **Get branch details**
   - Repository and branch name
   - View commit info, protection status

3. **Create branch**
   - Repository
   - New branch name
   - Source branch/commit SHA

4. **Delete branch**
   - Repository
   - Branch name

5. **Merge branches**
   - Repository
   - Base branch
   - Head branch
   - Commit message

6. **Compare branches**
   - Repository
   - Base branch
   - Compare branch
   - View diff and commits

7. **Get branch protection**
   - Repository
   - Branch name
   - View protection rules

8. **Update branch protection**
   - Repository
   - Branch name
   - Protection settings:
     * Require pull request reviews
     * Required status checks
     * Enforce for admins
     * Require linear history
     * Allow force pushes
     * Allow deletions

9. **Remove branch protection**
   - Repository
   - Branch name

10. **Rename branch**
    - Repository
    - Old branch name
    - New branch name

11. **Sync branch**
    - Repository
    - Branch name
    - Sync with upstream

12. **Get default branch**
    - Repository
    - View default branch name

13. **Set default branch**
    - Repository
    - Branch name
