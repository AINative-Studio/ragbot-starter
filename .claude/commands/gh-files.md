# GitHub File Operations

Available operations:

1. **Get file content**
   - Repository (owner/repo)
   - File path
   - Branch/commit reference (optional)

2. **Get directory contents**
   - Repository
   - Directory path
   - Branch/commit reference (optional)

3. **Create file**
   - Repository
   - File path
   - File content
   - Commit message
   - Branch

4. **Update file**
   - Repository
   - File path
   - New content
   - Commit message
   - File SHA (for conflict detection)
   - Branch

5. **Delete file**
   - Repository
   - File path
   - Commit message
   - File SHA
   - Branch

6. **Create or update file**
   - Repository
   - File path
   - Content
   - Commit message
   - Branch
   - Auto-detects create vs update

7. **Get file metadata**
   - Repository
   - File path
   - View size, SHA, type, links

8. **Download file**
   - Repository
   - File path
   - Save to local path

9. **Upload file**
   - Repository
   - File path
   - Local file to upload
   - Commit message

10. **Get repository archive**
    - Repository
    - Format (zipball, tarball)
    - Reference (branch/tag)

11. **Get raw file content**
    - Repository
    - File path
    - Reference

12. **Search files**
    - Repository
    - Query string
    - Path filter

13. **Get file history**
    - Repository
    - File path
    - View commit history for specific file
