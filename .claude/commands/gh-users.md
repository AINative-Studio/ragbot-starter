# GitHub User & Organization Management

Available operations:

## User Operations
1. **Get authenticated user**
   - View your own profile
   - Email, plan, permissions

2. **Get user by username**
   - View any public user profile
   - Bio, location, company, website

3. **List user repositories**
   - Username
   - Filter by type, sort options

4. **List user gists**
   - Username
   - Public gists

5. **List user followers**
   - Username
   - View all followers

6. **List user following**
   - Username
   - View who user follows

7. **Check if user follows another**
   - Target username

8. **Follow user**
   - Username to follow

9. **Unfollow user**
   - Username to unfollow

10. **List user organizations**
    - Username
    - View org memberships

11. **List user starred repositories**
    - Username
    - Repositories user has starred

12. **Star repository**
    - Repository (owner/repo)

13. **Unstar repository**
    - Repository

14. **Check if repository is starred**
    - Repository

## Organization Operations
15. **Get organization**
    - Organization name
    - View org profile

16. **List organization repositories**
    - Organization name
    - Type (all, public, private, forks, sources, member)

17. **List organization members**
    - Organization name
    - Role filter (all, admin, member)

18. **Check organization membership**
    - Organization name
    - Username

19. **List organization teams**
    - Organization name

20. **Get team details**
    - Organization name
    - Team slug

21. **List team members**
    - Organization name
    - Team slug

22. **List organization projects**
    - Organization name
    - State filter
